# Welcome to My Blog!

This is a sample blog post to help you get started. You can edit this content directly through [Pages CMS](https://pagescms.org) or by editing the markdown files in `src/content/blog/`.

## How to Create a New Post

1. Go to [Pages CMS](https://pagescms.org) and sign in with your GitHub account
2. Navigate to your repository
3. Click on "Blog Posts" in the content section
4. Create a new post with title, excerpt, and content

## Frontmatter Fields

Each blog post supports the following fields:

- **title** (required): The post title
- **excerpt**: A short summary shown in previews
- **publishedAt** (required): Publication date
- **updatedAt**: Last update date
- **featured**: Mark as a featured post for homepage highlighting
- **tags**: Array of tags for categorization
- **cover**: Cover image with `src` and `alt` properties
- **draft**: Set to `true` to hide the post from production

## Markdown Features

You can use all standard markdown features:

- **Bold text** and *italic text*
- [Links](https://example.com)
- Lists (like this one!)
- Code blocks
- And more...

Happy blogging!
